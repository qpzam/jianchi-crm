## 二、大股东减持规定

### 2.1 大股东身份认定
- 持股5%以上的股东
- 实际控制人（无论持股是否达5%）
- 合并计算多个账户的持股、转融通出借未归还的股份等

### 2.2 减持比例限制汇总表

| 减持方式 | 90日内减持上限 | 受让方锁定期 |
|---------|--------------|-------------|
| 集中竞价 | ≤公司股份总数1% | - |
| 大宗交易 | ≤公司股份总数2% | 受让后6个月 |
| 协议转让 | 单次受让方≥5% | 受让后6个月 |

### 2.3 减持计划披露
- 首次卖出前15个交易日报告并披露
- 披露内容：数量、来源、时间区间、价格区间、方式、原因

## 三、董监高减持规定

### 3.1 年度减持比例限制
- 每年减持不得超过其所持本公司股份总数的25%
- 持股不超过1000股的，可一次全部转让

### 3.2 不得转让情形

| 情形 | 限制期限 |
|-----|---------|
| 股票上市交易之日起 | 1年内 |
| 离职后 | 半年内 |
| 涉嫌违法犯罪被调查/处罚/判刑 | 未满6个月 |
| 被交易所公开谴责 | 未满3个月 |

### 3.3 窗口期限制
- 年度/半年度报告公告前15日内
- 季报、业绩预告、业绩快报公告前5日内
- 重大事项发生之日至披露之日

## 四、特定股东减持规定

### 4.1 适用主体
- 大股东以外持有IPO前股份的股东（特定股东）
- 非公开发行取得的股东

### 4.2 减持比例限制

| 减持方式 | 减持上限 |
|---------|---------|
| 集中竞价 | 90日内≤公司股份总数1% |
| 大宗交易 | 90日内≤公司股份总数2% |

---

## 五、创投基金减持特殊规定

### 5.1 适用条件
- 在中基协备案的创业投资基金
- 投资企业满足早期企业/中小企业/高新技术企业标准

### 5.2 减持额度阶梯表

| 投资期限 | 集中竞价 | 大宗交易 |
|---------|---------|---------|
| <36个月 | 90日内≤1% | 90日内≤2% |
| ≥36个月<48个月 | 60日内≤1% | 60日内≤2% |
| ≥48个月<60个月 | 30日内≤1% | 30日内≤2% |
| ≥60个月 | 不受限制 | 不受限制 |

### 5.3 大宗交易受让方锁定期
- 投资期限<60个月：受让方6个月内不得减持
- 投资期限≥60个月：**豁免受让方6个月锁定期**

### 6.1 集中竞价交易

### 6.3 协议转让

| 项目 | 要求 |
|-----|------|
| 单次受让方比例 | 不得低于5% |
| 受让方锁定期 | 6个月内不得减持 |

## 七、不得减持的情形

### 7.1 大股东不得减持

| 情形 | 限制期限 |
|-----|---------|
| 涉嫌证券期货违法犯罪被调查/侦查/处罚/判刑 | 未满6个月 |
| 被交易所公开谴责 | 未满3个月 |
| 尚未足额缴纳罚没款 | 持续（减持资金用于缴纳除外） |

### 7.2 控股股东/实控人不得减持

| 情形 | 限制期限 |
|-----|---------|
| 上市公司涉嫌违法犯罪被调查/处罚/判刑 | 未满6个月 |
| 上市公司被公开谴责 | 未满3个月 |
| 可能触及重大违法强制退市情形 | 限制转让期限内 |

## 八、破发破净分红不达标限制

### 8.1 限制类型

| 限制类型 | 条件 | 受限主体 | 减持方式 |
|---------|------|---------|---------|
| 破发 | 最近20个交易日任一日股价（向后复权）< IPO发行价 | IPO时控股股东/实控人及一致行动人 | 集中竞价、大宗交易 |
| 破净 | 最近20个交易日任一日股价 < 每股净资产 | 控股股东/实控人及一致行动人 | 集中竞价、大宗交易 |
| 分红不达标 | 最近3个年度未实施现金分红或累计分红<年均净利润30% | 控股股东/实控人及一致行动人 | 集中竞价、大宗交易 |

### 8.2 适用主体差异
- 控股股东/实控人及其一致行动人：全部适用
- 非控股大股东：仅当为持股5%以上第一大股东时比照执行
- 创投基金：如非控股股东/实控人身份，不受限制

## 九、减持信息披露要求

### 9.1 减持计划披露

| 项目 | 要求 |
|-----|------|
| 披露时间 | 首次卖出前15个交易日 |
| 披露内容 | 数量、来源、时间区间、价格区间、方式、原因 |
| 计划期间 | 最长3个月 |

### 9.2 减持进展披露

| 情形 | 披露时间 |
|-----|---------|
| 实施完毕 | 2个交易日内 |
| 计划期限届满 | 2个交易日内 |
| 发生重大事项 | 同步披露进展 |

### 9.3 股份变动披露

| 主体 | 披露时间 |
|-----|---------|
| 董监高 | 事实发生日起2个交易日内 |
| 5%以上股东 | 权益变动5%时报告并公告 |

## 十一、股份锁定期安排

### 11.1 IPO股份锁定期

| 股份类型 | 持有人 | 锁定期 |
|---------|-------|--------|
| 控股股东、实控人 | - | 36个月 |
| 突击入股（主板） | 上市前一年从控股/实控人受让或增资 | 36个月 |
| 突击入股（创业板） | 申请受理前6个月从控股/实控人受让或增资 | 36个月 |
| 发起人及发行前股份 | - | 1年 |

### 11.2 非公开发行锁定期

| 持有类型 | 持有人 | 锁定期 |
|---------|-------|--------|
| 控股股东/实控人、战略投资者 | - | 36个月 |
| 其他发行对象 | - | 12个月 |
| 实控人/通过认购取得实际控制权 | - | 36个月 |

### 11.3 重大资产重组锁定期

| 情形 | 锁定期 |
|-----|--------|
| 特定对象为控股股东/实控人 | 16个月 |
| 特定对象为私募投资基金 | 持足18个月 |
| 其他特定对象 | 12个月 |

## 十二、大宗交易规则

### 12.1 大宗交易条件

| 证券类型 | 数量或金额条件 |
|---------|---------------|
| A股 | ≥30万股 或 ≥200万元 |
| B股 | ≥3万股 或 ≥20万元港币 |
| 基金 | ≥200万份 或 ≥200万元 |
| 债券 | ≥5千张 或 ≥50万元 |

### 12.2 交易方式

| 方式 | 说明 |
|-----|------|
| 协议大宗交易 | 双方互为指定交易对手方，协商确定价格和数量 |
| 盘后定价大宗交易 | 按收盘价或成交量加权平均价撮合 |

### 12.3 交易时间

| 方式 | 申报时间 |
|-----|---------|
| 协议大宗 | 9:15-11:30、13:00-15:30 |
| 盘后定价 | 15:05-15:30 |

## 十三、协议转让规则

### 13.1 单次受让比例要求
- 单个受让方受让比例不得低于公司股份总数的5%

### 13.2 受让方锁定期
- 受让方在受让后6个月内不得减持

### 13.3 出让方后续义务
- 大股东减持后不再具有大股东身份的：减持后6个月内仍需遵守相关规定
- 控股股东/实控人减持后不再具有上述身份的：减持后6个月内仍需遵守相关规定

## 十五、创业板询价转让配售

### 15.1 询价转让流程

| 步骤 | 时间 | 操作 |
|-----|------|------|
| 第一步 | 计划书披露日15:00前 | 股东通过托管券商进行股份锁定申报 |
| 第二步 | 当日日终 | 中国结算实施股份锁定 |
| 第三步 | T日15:00前 | 受托券商申报成交结果 |
| 第四步 | T日日终 | 解除锁定+办理过户登记 |
| 第五步 | T+1日 | 扣收税费 |

### 15.2 配售流程

| 步骤 | 时间 | 操作 |
|-----|------|------|
| 第一步 | 计划书披露日前一交易日收市后 | 受托券商申报配售登记信息 |
| 第二步 | 计划书披露日15:00前 | 股东进行配售股份锁定申报 |
| 第三步 | 配售股权登记日 | 中国结算向其他股东分配配售权 |
| 第四步 | 配售登记日+5个交易日（T日） | 配售对象申购配售股份 |
| 第五步 | T日日终 | 解除锁定+办理过户登记 |

## 附录三：常见问题解答

### Q1：如何确定大股东身份？
A：持股5%以上的股东、实际控制人（无论持股是否达5%），合并计算多个账户、转融通出借等股份。

### Q2：减持至低于5%后是否还有限制？
A：自持股比例降至低于5%之日起90日内，仍需遵守大股东减持规定。

### Q3：创投基金是否受破发、破净、分红限制？
A：如非控股股东/实控人身份，不受破发、破净、分红不达标限制。

### Q4：离婚后股份如何处理？
A：股份过出方、过入方应当持续共同遵守减持规定，控股股东/实控人需持续遵守更严格规定。

### Q5：司法强制执行如何处理？
A：根据具体减持方式分别适用相关规定，收到执行通知后2个交易日内披露。
