# 第2批：交易所规则-深交所 - 规则摘要

## 一、深交所自律监管指引第18号

文件：《深圳证券交易所上市公司自律监管指引第18号——股东及董事、监事、高级管理人员减持股份》

---

## 二、核心减持规则

### 2.1 大股东减持规则（第四条至第八条）

**定义**：持股5%以上股东

**减持比例限制**：
- **集中竞价交易**：在任意连续90个自然日内，减持股份总数不得超过公司股份总数的1%
- **大宗交易方式**：在任意连续90个自然日内，减持股份总数不得超过公司股份总数的2%

**大宗交易受让方限制**：
- 受让方在受让后6个月内，不得转让其受让的股份

**协议转让限制**：
- 单个受让方的受让比例不得低于公司股份总数的5%

### 2.2 控股股东/实际控制人减持规则（第九条至第十条）

**特殊限制**：
- IPO后36个月内，控股股东、实际控制人及其一致行动人不得转让其持有的公司股份

**减持限制触发条件（"三不"情形）**：
当存在以下任一情形时，控股股东、实际控制人不得通过二级市场减持：
1. 破发：公司股票收盘价低于IPO发行价
2. 破净：公司股票收盘价低于最近一个会计年度或者最近一期财务会计报告披露的每股净资产
3. 分红不达标：最近三年未进行现金分红，或累计现金分红金额低于最近三年年均净利润的30%

**计算基准**：
- 上市未满三年的，以首次公开发行上市日为基准
- 上市满三年的，以最近一个完整的会计年度经审计的财务会计报告为基准

### 2.3 董监高减持规则（第十一条）

**年度减持上限**：
- 每年转让的股份不得超过其所持有本公司股份总数的25%

**离职后限制**：
- 离职后半年内，不得转让其所持有的本公司股份

**短线交易禁止**：
- 将其持有的该公司的股票在买入后六个月内卖出，或者在卖出后六个月内又买入，由此所得收益归该公司所有

### 2.4 特殊股东减持规则

**第一大股东但不是控股股东**：减持时参照控股股东、实际控制人执行

**实际控制人的一致行动人**：减持时参照控股股东、实际控制人执行

## 三、预披露要求（第三章）

### 3.1 大股东预披露（第十二条）

**集中竞价减持**：
- 应当在首次卖出前15个交易日预披露减持计划

**减持计划内容**：
- 拟减持股份的数量
- 拟减持股份的来源
- 减持时间区间
- 减持方式
- 减持价格区间（如有）
- 减持原因
- 本所规定的其他内容

**时间要求**：
- 减持时间区间不得超过6个月

**变更要求**：
- 减持计划实施期间，拟变更减持时间区间、方式等的，应当重新披露减持计划

### 3.2 减持期间信息披露（第十三条）

**减持进展披露**：
- 在减持时间区间过半时，应当披露减持进展情况

**减持比例达到1%披露**：
- 减持比例达到公司股份总数1%时，应当在事实发生之日起两个交易日内披露

**减持结果披露**：
- 减持计划实施完毕后，应当在两个交易日内予以公告
- 减持计划期限届满时，如未实施减持或减持未实施完毕，也应当予以公告

### 3.3 董监高预披露（第十四条）

**要求**：
- 董监高通过集中竞价交易减持的，参照大股东预披露规定执行

### 3.4 协议转让披露（第十五条）

**披露要求**：
- 股份通过协议转让方式减持的，单个受让方的受让比例不得低于公司股份总数的5%
- 转让双方应当在交易达成后及时披露相关信息

## 四、减持限制（第十六条）

**暂停减持情形**：
具有下列情形之一的，大股东及董监高不得减持公司股份：
1. 上市公司或者大股东、董监高因涉嫌证券期货违法犯罪，在被中国证监会立案调查或者被司法机关立案侦查期间
2. 大股东、董监高被中国证监会行政处罚，或者被交易所公开谴责未满3个月的
3. 上市公司因涉嫌证券期货违法犯罪，在被中国证监会立案调查或者被司法机关立案侦查期间，以及被中国证监会行政处罚、司法机关判决有罪未满6个月的
4. 上市公司触及重大违法强制退市标准的，自相关行政处罚决定或者司法裁判作出之日起至公司股票终止上市前

---

## 五、豁免情形（第十七条至第十八条）

### 5.1 预披露豁免（第十七条）

以下情形可豁免预披露：
1. 因公司发行股份、可转换公司债券等导致持股比例被动增加后减持
2. 因司法强制执行、股权激励行权等原因减持股份
3. 因继承、赠与、协议分割等非交易方式转让股份
4. 因实施股权激励计划、员工持股计划导致董监高减持股份
5. 董监高因可交换公司债券换股、股权激励行权、认购或申购ETF等减持股份
6. 上市公司回购股份导致股东持股比例被动增加

### 5.2 比例限制豁免（第十八条）

以下情形可豁免减持比例限制：
1. 转让给配偶、子女
2. 转让给符合条件的创业投资基金股东
3. 通过协议转让方式转让

## 六、深交所交易规则相关条款

### 6.1 大宗交易规则（第三章第四节）

**大宗交易的申报数量**：
- A股单笔申报数量不低于30万股，或者交易金额不低于200万元人民币
- 基金单笔申报数量不低于200万份，或者交易金额不低于200万元人民币
- 债券单笔申报数量不低于1万张，或者交易金额不低于100万元人民币

**大宗交易价格范围**：
- 证券申报价格在该证券当日涨跌幅限制价格区间内
- 无涨跌幅限制的，由买卖双方在前收盘价的上下30%自行协商确定

### 6.2 协议转让规则

**协议转让要求**：
- 单个受让方的受让比例不得低于公司股份总数的5%
- 转让价格下限不得低于协议签署日前一交易日公司股票收盘价的90%
- 存在多种价格时，取孰低原则

**涉及国有资产的**：需同时遵守国资监管规定

**涉及外资的**：需同时遵守外商投资相关规定

### 6.3 收购规则

**权益变动报告**：
- 投资者持有或者通过协议、其他安排与他人共同持有一个上市公司已发行的有表决权股份达到5%时，应编制简式权益变动报告书并公告
- 持股比例每增加或者减少5%，应编制简式或详式权益变动报告书并公告

**要约收购**：
- 投资者持有或者通过协议、其他安排与他人共同持有一个上市公司已发行的有表决权股份达到30%时，继续增持股份的，应当依法向该上市公司所有股东发出要约

## 七、深交所减持规则总结

### 7.1 减持方式及限制

| 减持方式 | 适用主体 | 限制比例 | 时间计算 |
|---------|---------|---------|---------|
| 集中竞价 | 大股东/董监高 | 任意连续90日不超过1% | 按自然日计算 |
| 大宗交易 | 大股东/董监高 | 任意连续90日不超过2% | 按自然日计算 |
| 协议转让 | 所有股东 | 单次受让方≥5% | - |
| 询价转让 | 创业板/科创板IPO前股东 | 无比例限制 | - |

### 7.2 特殊情形处理

| 情形 | 处理方式 |
|-----|---------|
| 因赠与导致持股比例变化 | 不受减持比例限制，无需预披露 |
| 因司法执行减持 | 不受减持比例限制，需及时公告 |
| 因股权激励行权 | 不受减持比例限制，需及时公告 |
| 转让给配偶/子女 | 可豁免预披露和比例限制 |
| 上市公司回购 | 持股比例被动变化，不属于减持 |

### 7.3 关键时间节点

| 事项 | 时间要求 |
|-----|---------|
| 大股东预披露 | 首次卖出前15个交易日 |
| 减持进展披露 | 减持时间区间过半时 |
| 减持比例1%披露 | 达到1%后2个交易日内 |
| 减持结果披露 | 实施完毕或期限届满后2个交易日 |
| 大宗交易受让方锁定期 | 受让后6个月内 |
| 董监高离职后锁定期 | 离职后6个月 |
