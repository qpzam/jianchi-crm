# 第6批：询价转让/配售方式减持 - 规则摘要

## 一、核心法规

### 1.1 法规名称
- 《深圳证券交易所创业板上市公司股东询价转让和配售方式转让股份业务指引》
- 《深圳证券交易所上市公司自律监管指引第16号——创业板上市公司股东询价转让和配售方式减持股份》

### 1.2 适用范围
- 在深圳证券交易所创业板上市的公司
- 持有该公司首次公开发行前股份的股东

### 1.3 业务类型
1. 询价转让
2. 配售方式

---

## 二、询价转让规则

### 2.1 询价转让定义

询价转让，是指上市公司IPO前股东向符合条件的机构投资者询价转让其所持股份的行为。

### 2.2 适用主体

**转让方**：
- 创业板上市公司IPO前股东
- 包括控股股东、实际控制人及其一致行动人

**受让方**：
- 符合条件的机构投资者
- 包括基金管理公司、证券公司、信托公司、财务公司、保险公司、合格境外机构投资者等

### 2.3 询价转让优势

| 优势 | 说明 |
|-----|------|
| 不受减持比例限制 | 不受集中竞价1%、大宗交易2%的限制 |
| 价格市场化 | 通过询价确定转让价格 |
| 不影响二级市场 | 不通过二级市场交易，不直接冲击股价 |
| 可锁定退出价格 | 可提前锁定减持价格 |
| 批量转让 | 可一次性转让较大数量股份 |

### 2.4 询价转让流程

**第一步：发布公告**
- 转让方披露询价转让计划书
- 内容包括：拟转让股份的数量、价格区间、转让原因等

**第二步：组织询价**
- 通过中介机构向符合条件的机构投资者询价
- 收集报价和申购数量

**第三步：确定受让方**
- 根据报价和申购数量确定受让方
- 确定转让价格

**第四步：完成转让**
- 通过中国结算办理股份过户
- 披露转让结果

### 2.5 询价转让限制

**转让数量限制**：
- 单次询价转让数量不得低于公司股份总数的1%
- 单次询价转让数量不得超过公司股份总数的5%

**转让频率限制**：
- 每个交易日不超过1次询价转让
- 每月询价转让次数有上限

**价格限制**：
- 转让价格不得低于转让公告日前一交易日收盘价的80%
- 转让价格不得低于转让公告日前20个交易日均价的90%

## 三、配售方式规则

### 3.1 配售方式定义

配售方式，是指上市公司股东向符合条件的机构投资者配售其所持股份的行为。

### 3.2 适用主体

**转让方**：
- 创业板上市公司股东

**受让方**：
- 符合条件的机构投资者
- 与询价转让受让方范围一致

### 3.3 配售方式优势

| 优势 | 说明 |
|-----|------|
| 不受减持比例限制 | 不受集中竞价1%、大宗交易2%的限制 |
| 锁定受让方 | 可预先确定受让方 |
| 定价灵活 | 可协商确定转让价格 |
| 不影响二级市场 | 不通过二级市场交易 |

### 3.4 配售方式流程

**第一步：确定配售方案**
- 转让方确定配售股份的数量、价格、受让方等

**第二步：发布公告**
- 披露配售方案
- 内容包括：配售股份的数量、价格、受让方、配售原因等

**第三步：签署协议**
- 转让方与受让方签署配售协议

**第四步：完成配售**
- 通过中国结算办理股份过户
- 披露配售结果

### 3.5 配售方式限制

**配售数量限制**：
- 单次配售数量不得低于公司股份总数的1%
- 单次配售数量不得超过公司股份总数的5%

**受让方数量限制**：
- 受让方数量不少于10家

**价格限制**：
- 配售价格不得低于配售公告日前一交易日收盘价的80%
- 配售价格不得低于配售公告日前20个交易日均价的90%

## 四、询价转让与配售方式对比

### 4.1 主要区别

| 项目 | 询价转让 | 配售方式 |
|-----|---------|---------|
| 受让方确定方式 | 通过询价确定 | 预先确定 |
| 价格确定方式 | 询价确定 | 协商确定 |
| 透明度 | 较高 | 相对较低 |
| 适用场景 | 希望市场化定价 | 希望定向转让 |

### 4.2 共同特点

| 项目 | 规则 |
|-----|------|
| 不受减持比例限制 | 不受集中竞价1%、大宗交易2%的限制 |
| 受让方锁定 | 受让方6个月内不得转让 |
| 价格下限 | 不得低于前日收盘价80% |
| 不影响二级市场 | 不通过二级市场交易 |

---

## 五、询价转让/配售方式信息披露

### 5.1 计划披露

**询价转让**：
- 应当在询价开始前披露询价转让计划书
- 内容包括：拟转让股份的数量、价格区间、转让原因、询价时间等

**配售方式**：
- 应当在配售开始前披露配售方案
- 内容包括：配售股份的数量、价格、受让方、配售原因等

### 5.2 进展披露

**询价转让**：
- 询价结束后2个交易日内披露询价结果
- 包括报价情况、申购数量、最高报价等

**配售方式**：
- 配售完成后2个交易日内披露配售结果

### 5.3 结果披露

**询价转让/配售方式**：
- 转让完成后2个交易日内披露转让结果
- 内容包括：转让股份数量、价格、受让方、转让金额等

## 六、询价转让/配售方式与其他减持方式对比

### 6.1 减持方式对比

| 减持方式 | 适用主体 | 减持比例限制 | 是否影响二级市场 |
|---------|---------|-------------|----------------|
| 集中竞价 | 所有股东 | 90日不超过1% | 直接影响 |
| 大宗交易 | 所有股东 | 90日不超过2% | 间接影响 |
| 协议转让 | 所有股东 | 单次≥5% | 不直接影响 |
| 询价转让 | 创业板/科创板IPO前股东 | 无限制 | 不影响 |
| 配售方式 | 创业板/科创板股东 | 无限制 | 不影响 |

### 6.2 选择建议

| 减持需求 | 建议方式 |
|---------|---------|
| 小量减持 | 集中竞价 |
| 中量减持 | 大宗交易 |
| 大量减持 | 询价转让/协议转让 |
| 不想影响股价 | 询价转让/配售方式/协议转让 |
| 创业板/科创板股东 | 询价转让/配售方式 |

## 七、询价转让/配售方式实操要点

### 7.1 适用条件

**IPO前股东**：
- 必须是IPO前股东
- 包括控股股东、实际控制人及其一致行动人

**交易所**：
- 目前适用于创业板、科创板
- 主板暂不适用

**受让方**：
- 必须是符合条件的机构投资者
- 包括公募基金、私募基金、券商、保险等

### 7.2 时间节点

| 节点 | 时间要求 |
|-----|---------|
| 发布公告 | 询价/配售前 |
| 询价结果披露 | 询价结束后2个交易日内 |
| 完成转让披露 | 转让完成后2个交易日内 |

### 7.3 价格确定

**询价转让**：
- 通过询价确定
- 遵循"价格优先、数量优先"原则

**配售方式**：
- 协商确定
- 不得低于价格下限

### 7.4 过户流程

**询价转让/配售方式**：
- 通过中国结算办理过户
- 需要中介机构参与

### 8.1 减持便利性

| 优势 | 说明 |
|-----|------|
| 不受减持比例限制 | 可一次性大量减持 |
| 不影响二级市场 | 不直接冲击股价 |
| 可锁定价格 | 提前锁定减持价格 |

### 8.2 与业务合作

- 创业板、科创板IPO前股东
- 减持量大的股东
- 对股价敏感的股东

**合作价值**：
- 提供询价转让服务
- 提供配售方案设计
- 协助联系机构投资者

## 九、询价转让/配售方式与创投基金减持的结合

### 9.1 创投基金的优势

创投基金股东往往持有IPO前股份，可利用询价转让/配售方式减持：
1. 创投基金投资期限长，减持意愿强
2. 通过询价转让可快速退出
3. 不受减持比例限制

### 9.2 适用情形

| 情形 | 建议方式 |
|-----|---------|
| 创投基金投资36个月以上 | 询价转让/配售方式 |
| 创投基金减持量大 | 询价转让/配售方式 |
| 创投基金不想影响股价 | 询价转让/配售方式 |

| 要求 | 说明 |

## 十三、询价转让/配售方式实操案例

### 13.1 案例特点

**询价转让成功案例特点**：
1. 股票流动性较好
2. 机构投资者参与度高
3. 转让价格合理

**配售方式成功案例特点**：
1. 提前确定了受让方
2. 价格协商成功
3. 信息披露充分

### 13.2 案例启示

1. 选择合适的时机
2. 做好信息披露
3. 与机构投资者充分沟通
4. 选择有经验的中介机构
