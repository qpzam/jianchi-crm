# 第4批：董监高规则 - 规则摘要

## 一、董监高股份管理规则（2022年修订）

文件：《上市公司董事、监事和高级管理人员所持本公司股份及其变动管理规则（2022年修订）》
制定单位：中国证券监督管理委员会、国务院国有资产监督管理委员会

---

## 二、董监高定义与范围

### 2.1 董事、监事、高级管理人员范围

**董事**：
- 包括独立董事、非独立董事

**监事**：
- 包括股东代表监事、职工代表监事

**高级管理人员**：
- 总经理、副总经理、财务负责人、董事会秘书
- 公司章程规定的其他高级管理人员

### 2.2 适用范围

本规则适用于：
- 在证券交易所上市的公司
- 上述公司中担任董事、监事、高级管理职务的人员
- 虽已离职但仍受限制的董事、监事、高级管理人员

---

## 三、董监高股份变动核心规则

### 3.1 禁止短线交易（第六条）

**规定内容**：
- 董监高将其持有的该公司的股票在买入后六个月内卖出，或者在卖出后六个月内又买入，由此所得收益归该公司所有
- 董事会应当收回其所得收益

**收益计算**：
- 收益=（卖出价-买入价）×数量
- 如买入价高于卖出价，差价部分仍需上缴

**违规后果**：
- 收益归公司所有
- 公司董事会未收回的，股东有权要求董事会收回
- 董事会拒不收回的，股东有权提起诉讼

### 3.2 年度减持上限（第七条）

**规定内容**：
- 董监高每年转让的股份不得超过其所持有本公司股份总数的25%

**基数计算**：
- 以其上年末所持有本公司发行的股份总数为基数
- 上年末持股不足1000股的，可以一次性转让

**例外情形**：
- 因司法强制执行、继承、赠与等被动增加的股份，不计入基数

### 3.3 离职后限制（第八条）

**规定内容**：
- 董监高离职后半年内，不得转让其所持有的本公司股份

**离职认定**：
- 以董事会决议或股东大会决议生效之日起计算

**适用范围**：
- 适用于因辞职、被解聘、任期届满等原因离职的董监高

### 3.4 新增股份锁定（第九条）

**规定内容**：
- 董监高在任职期间，每年转让的股份不得超过其所持有本公司股份总数的25%
- 新增股份（如送股、转增、配股等），在12个月内不得转让

**锁定起算**：
- 股权登记日或发行完成日

### 3.5 信息披露要求（第十条）

**规定内容**：
- 董监高买卖本公司股票后，应当在两个交易日内公告

**公告内容**：
- 买卖的股份数量
- 买卖的价格
- 买卖的日期
- 买卖后的持股数量

### 3.6 窗口期禁止买卖（第十一条）

**规定内容**：
- 在上市公司定期报告公告前30日内，不得买卖本公司股票
- 在业绩预告、业绩快报公告前10日内，不得买卖本公司股票
- 在可能对本公司股票及其衍生品种交易价格产生较大影响的重大事项发生之日或在决策过程中，至依法披露后2个交易日内，不得买卖本公司股票

**重大事项包括**：
- 重大投资、重大资产重组
- 重大合同签订
- 重大人事变动
- 重大技术突破等

---

## 四、董监高减持比例计算

### 4.1 基数确定

**基数构成**：
- 以其上年末所持有本公司发行的股份总数为基数
- 包括首发前股份、定增股份、股权激励股份等

**基数调整**：
- 因送股、转增、配股等增加的股份，计入基数
- 因公司回购减少的股份，相应减少基数

### 4.2 计算示例

**示例1**：
- 董监高上年末持有100万股公司股份
- 当年可减持上限=100万股×25%=25万股

**示例2**：
- 董监高上年末持有500股公司股份
- 因不足1000股，可一次性转让全部500股

### 4.3 与其他规则叠加

**董监高同时是大股东**：
- 需同时遵守董监高减持规则和大股东减持规则
- 年度减持不超过25%，且90日内集中竞价不超过1%

**董监高是控股股东**：
- 需同时遵守控股股东特殊限制
- IPO后36个月不得转让，破发、破净、分红不达标不得减持

## 五、董监高股份变动禁止情形

### 5.1 绝对禁止情形

| 情形 | 禁止内容 |
|-----|---------|
| 短线交易 | 买入后6个月内卖出，或卖出后6个月内买入 |
| 窗口期 | 定期报告前30日、业绩预告前10日买卖股票 |
| 重大事项敏感期 | 重大事项决策期间至披露后2个交易日内买卖股票 |
| 离职后半年内 | 转让所持有的公司股份 |

### 5.2 相对禁止情形

| 情形 | 限制内容 |
|-----|---------|
| 年度减持 | 不超过持股总数的25% |
| 90日集中竞价 | 不超过公司股份总数的1% |
| 新增股份 | 12个月内不得转让 |

---

## 六、违规减持后果

### 6.1 收益追缴

| 违规情形 | 后果 |
|---------|------|
| 短线交易 | 收益归公司所有 |
| 窗口期买卖 | 收益可能被追缴 |
| 其他违规 | 监管措施 |

### 6.2 监管措施

| 措施 | 适用情形 |
|-----|---------|
| 监管谈话 | 一般违规 |
| 书面警示 | 一般违规 |
| 通报批评 | 较严重违规 |
| 公开谴责 | 严重违规 |

### 6.3 法律责任

- 违反《证券法》规定的，依法承担法律责任
- 情节严重的，可能构成刑事犯罪

---

## 七、董监高与普通股东减持对比

### 7.1 主要区别

| 项目 | 董监高 | 普通股东 |
|-----|-------|---------|
| 年度减持上限 | 持股25% | 无限制 |
| 离职后锁定 | 半年 | 无 |
| 窗口期限制 | 有 | 无 |
| 短线交易 | 禁止 | 无特殊限制 |
| 预披露 | 需要（集中竞价） | 大股东需要 |

### 7.2 共同规则

| 项目 | 规则 |
|-----|------|
| 90日集中竞价 | 不超过公司股份总数1% |
| 预披露 | 首次卖出前15个交易日 |
| 禁止内幕交易 | 适用于所有主体 |
| 禁止操纵市场 | 适用于所有主体 |

## 八、实操要点

### 8.1 时间计算

**年度计算**：
- 以公历年度为计算周期
- 从1月1日至12月31日

**90日计算**：
- 按连续自然日计算
- 包括交易日和非交易日

**窗口期计算**：
- 定期报告：公告前30个自然日
- 业绩预告：公告前10个自然日

### 8.2 比例计算

**25%计算**：
- 基数=上年末持股总数
- 可减持数量=基数×25%

**1%计算**：
- 基数=公司股份总数
- 可减持数量=公司股份总数×1%

### 8.3 信息披露

**交易后披露**：
- 交易完成后2个交易日内公告

**内容要求**：
- 股份数量、价格、日期、持股变化

---

## 九、董监高减持规则总结

### 9.1 核心限制

| 限制类型 | 具体要求 |
|---------|---------|
| 年度减持 | 不超过持股总数25% |
| 90日集中竞价 | 不超过公司股份总数1% |
| 短线交易 | 禁止（6个月内反向交易） |
| 窗口期买卖 | 禁止（定期报告前30日等） |
| 离职后锁定 | 半年内不得转让 |

### 9.2 信息披露

| 信息类型 | 披露时间 |
|---------|---------|
| 交易后公告 | 交易完成后2个交易日内 |
| 预披露公告 | 首次卖出前15个交易日 |
| 减持进展 | 减持时间区间过半时 |
| 减持结果 | 实施完毕或期限届满后2个交易日内 |

---

## 十、董监高与关联方减持

### 10.1 关联方范围

- 配偶、父母、子女
- 配偶的父母、子女的配偶
- 兄弟姐妹及其配偶、配偶的兄弟姐妹
- 控制的其他企业

### 10.2 关联方交易

- 关联方买卖公司股票的，董监高需承担相应责任
- 关联方与董监高合并计算持股比例

### 10.3 转让给关联方

- 转让给配偶、子女的，可豁免预披露和比例限制
- 但需符合其他监管要求
