# 第3批：交易所规则-上交所 - 规则摘要

## 一、上交所股份增减持规则（理解一本通）

文件：《上海证券交易所上市公司股份增减持规则理解一本通》

## 二、上交所减持规则框架

上交所的减持规则与深交所基本一致，均遵循证监会统一的减持规定框架：

### 2.1 规则层级

1. **法律层面**：《公司法》《证券法》
2. **行政法规**：《上市公司股权激励管理办法》等
3. **部门规章**：《上市公司股东、董监高减持股份的若干规定》（证监会公告〔2017〕9号）
4. **交易所规则**：《上海证券交易所上市公司股东及董事、监事、高级管理人员减持股份实施细则》

## 三、上交所核心减持规则

### 3.1 大股东减持规则

**定义**：持股5%以上股东

**减持比例限制**：

| 减持方式 | 限制比例 | 计算周期 |
|---------|---------|---------|
| 集中竞价交易 | 不超过公司股份总数的1% | 任意连续90个自然日 |
| 大宗交易 | 不超过公司股份总数的2% | 任意连续90个自然日 |
| 协议转让 | 单个受让方不低于5% | - |

**大宗交易受让方限制**：
- 受让方在受让后6个月内，不得转让其受让的股份

### 3.2 控股股东/实际控制人减持规则

**特殊限制**：

| 限制情形 | 具体要求 |
|---------|---------|
| IPO锁定 | IPO后36个月内不得转让 |
| 破发限制 | 股价低于IPO发行价时不得减持 |
| 破净限制 | 股价低于每股净资产时不得减持 |
| 分红限制 | 最近三年分红不达标时不得减持 |

**计算基准**：
- 破发：以IPO发行价格为基准
- 破净：以最近一个会计年度或最近一期财务报告披露的每股净资产为基准
- 分红：以最近三年累计现金分红与年均净利润的比例为基准（不低于30%）

### 3.3 董监高减持规则

**年度减持上限**：
- 每年转让的股份不得超过其所持有本公司股份总数的25%

**离职后限制**：
- 离职后半年内，不得转让其所持有的本公司股份

**任职期间新增股份**：
- 新增股份（如送股、转增、配股）在12个月内不得转让

### 3.4 第一大股东减持规则

**特别规定**：
- 第一大股东（持股比例最高）但不是控股股东的，减持时参照控股股东、实际控制人执行

---

## 四、预披露要求

### 4.1 大股东预披露

**要求**：
- 通过集中竞价交易减持的，应当在首次卖出前15个交易日预披露减持计划

**减持计划内容**：
- 拟减持股份的数量、来源
- 减持时间区间（最长不超过6个月）
- 减持方式、价格区间（如有）
- 减持原因
- 上交所规定的其他内容

### 4.2 减持期间信息披露

**减持进展**：
- 在减持时间区间过半时，应当披露减持进展情况

**减持比例达到1%**：
- 减持比例达到公司股份总数1%时，应当在事实发生之日起两个交易日内披露

**减持结果**：
- 减持计划实施完毕或期限届满后，应当在两个交易日内公告

### 4.3 董监高预披露

**要求**：
- 董监高通过集中竞价交易减持的，参照大股东预披露规定执行

---

## 五、减持限制情形

### 5.1 不得减持的情形

具有下列情形之一的，大股东及董监高不得减持公司股份：
1. 上市公司或大股东、董监高因涉嫌证券期货违法犯罪，在被证监会立案调查或被司法机关立案侦查期间
2. 大股东、董监高被证监会行政处罚，或被交易所公开谴责未满3个月的
3. 上市公司因涉嫌证券期货违法犯罪，在被证监会立案调查或被司法机关立案侦查期间，以及被证监会行政处罚、司法机关判决有罪未满6个月的
4. 上市公司触及重大违法强制退市标准的，自相关行政处罚决定或司法裁判作出之日起至公司股票终止上市前

### 5.2 暂停减持的情形

具有下列情形之一的，应暂停减持计划：
1. 上市公司发生重大事项
2. 上市公司股价异常波动
3. 监管部门要求暂停

---

## 六、豁免情形

### 6.1 预披露豁免

以下情形可豁免预披露：
1. 因公司发行股份、可转换公司债券等导致持股比例被动增加后减持
2. 因司法强制执行、股权激励行权等原因减持股份
3. 因继承、赠与、协议分割等非交易方式转让股份
4. 因实施股权激励计划、员工持股计划导致董监高减持股份
5. 董监高因可交换公司债券换股、股权激励行权、认购或申购ETF等减持股份
6. 上市公司回购股份导致股东持股比例被动增加

### 6.2 比例限制豁免

以下情形可豁免减持比例限制：
1. 转让给配偶、子女
2. 转让给符合条件的创业投资基金股东
3. 通过协议转让方式转让
4. 因司法强制执行减持

## 七、上交所市值管理规则相关

文件：《上海证券交易所上市公司市值管理规则理解一本通》

### 7.1 市值管理的定义

市值管理是指上市公司基于公司市值信号，综合运用多种经营管理方式，致力于公司价值创造、价值经营和价值实现，使公司市值与公司价值相匹配的经营管理活动。

### 7.2 市值管理与减持的关系

**合法合规**：
- 市值管理必须在法律法规框架内进行
- 不得利用内幕信息、操纵市场等非法手段进行市值管理

**减持配合**：
- 合法的市值管理可以为减持创造良好的市场环境
- 但市值管理不能成为规避减持监管的手段

### 7.3 合规要点

- 不得通过虚假信息披露影响股价
- 不得通过市场操纵影响股价
- 不得利用内幕信息进行交易
- 减持必须遵守预披露和减持比例限制

## 八、上交所与深交所规则对比

### 8.1 共同点

| 项目 | 规则 |
|-----|------|
| 大股东集中竞价减持 | 任意连续90日不超过1% |
| 大股东大宗交易减持 | 任意连续90日不超过2% |
| 控股股东IPO锁定 | IPO后36个月 |
| 预披露时间 | 首次卖出前15个交易日 |
| 董监高年度减持上限 | 持股总数的25% |
| 董监高离职后锁定 | 6个月 |

### 8.2 主要差异

| 项目 | 上交所 | 深交所 |
|-----|-------|-------|
| 规则名称 | 《减持实施细则》 | 《自律监管指引第18号》 |
| 监管措施 | 基本一致 | 基本一致 |
| 创投基金减持 | 基本一致 | 基本一致 |

**注**：上交所和深交所的减持规则核心内容基本一致，主要差异在于规则形式和个别细节。

## 九、实操要点

### 9.1 时间计算

**连续90个自然日**：
- 按自然日计算，非交易日也计入
- 不同减持方式分别计算

**减持时间区间**：
- 最长不超过6个月
- 从公告日开始计算

### 9.2 比例计算

**减持比例计算基数**：
- 按公司股份总数计算，不是按个人持股数量计算

**不同主体分别计算**：
- 控股股东、实际控制人及其一致行动人合并计算
- 不同董监高分别计算

### 9.3 预披露变更

**变更情形**：
- 减持时间区间变更
- 减持方式变更
- 减持数量变更
- 减持价格区间变更

**变更程序**：
- 应当重新披露减持计划

## 十、违规后果

### 10.1 行政处罚

| 违规情形 | 处罚措施 |
|---------|---------|
| 在限制转让期内转让证券 | 警告、没收违法所得、罚款 |
| 未按规定预披露 | 监管谈话、书面警示、通报批评 |
| 违反减持比例限制 | 监管谈话、书面警示、通报批评 |
| 短线交易 | 收益归公司、警告 |

### 10.2 民事责任

- 因违规减持给其他投资者造成损失的，应承担民事赔偿责任

### 10.3 刑事责任

- 情节严重的，可能构成操纵证券市场罪、内幕交易罪等
