# 上市公司股东减持规则摘要（批次11）

## 一、上市公司董事、监事和高级管理人员所持本公司股份及其变动管理规则

### 1.1 适用范围
- 上市公司董事、监事、高级管理人员
- 核心技术人员或核心业务人员等减持本公司股份的，还应当遵守本规则

### 1.2 所持股份定义
所持本公司股份包括：
- 登记在其名下的所有本公司股份
- 利用他人账户持有的所有本公司股份
- 融资融券交易中记载在其信用账户内的本公司股份

---

## 二、董监高不得转让股份的情形（第4条）

| 序号 | 情形 | 禁止期限 |
|-----|------|---------|
| 1 | 本公司股票上市交易之日起 | 1年内 |
| 2 | 本人离职后 | 半年内 |
| 3 | 上市公司涉嫌证券期货违法犯罪被立案调查/侦查/处罚/判刑 | 未满6个月 |
| 4 | 本人因涉嫌与上市公司有关的证券期货违法犯罪被立案调查/侦查/处罚/判刑 | 未满6个月 |
| 5 | 本人因证券期货违法被证监会行政处罚，尚未足额缴纳罚没款 | 减持资金用于缴纳罚没款的除外 |
| 6 | 本人因涉及与上市公司有关的违法违规被交易所公开谴责 | 未满3个月 |
| 7 | 上市公司可能触及重大违法强制退市情形 | 在限制转让期限内 |
| 8 | 其他法律法规、证监会和交易所规则及公司章程规定情形 | - |

---

## 三、年度减持比例限制（第5条）

### 3.1 基本规则
- 在就任时确定的任职期间，每年减持不得超过其所持本公司股份总数的25%
- 减持方式包括：集中竞价、大宗交易、协议转让等

### 3.2 例外情形
- 因司法强制执行、继承、遗赠、依法分割财产等导致股份变动的，不受25%限制

### 3.3 小额豁免
- 所持股份不超过1000股的，可一次全部转让

---

## 四、可转让股份数量计算（第6-7条）

### 4.1 计算基数
- 以上年末持有的本公司股份总数为基数

### 4.2 年内新增股份处理

| 新增股份类型 | 当年可转让比例 | 计入次年基数 |
|------------|--------------|------------|
| 新增无限售条件股份 | 25% | - |
| 新增有限售条件股份 | 不可转让 | 计入次年 |
| 权益分派导致股份增加 | 同比例增加 | - |

### 4.3 未转让股份结转
- 当年可转让但未转让的股份
- 计入当年末持有股份总数
- 作为次年可转让股份的计算基数

---

## 五、公司章程特别规定（第8条）
- 上市公司章程可以规定比本规则更长的限制转让期间
- 可以规定更低的可转让股份比例
- 可以附加其他限制转让条件

---

## 六、减持计划披露（第9条）

### 6.1 披露时间
- 首次卖出前**15个交易日**向交易所报告并披露

### 6.2 减持计划内容
| 内容项目 | 具体要求 |
|---------|---------|
| 拟减持股份的数量、来源 | - |
| 减持时间区间、价格区间、方式和原因 | 时间区间应符合交易所规定 |
| 不存在不得减持情形说明 | 涉及第4条情形 |
| 其他交易所规定内容 | - |

### 6.3 减持结果披露
- **实施完毕**：2个交易日内报告并公告
- **未实施或未完毕**：时间区间届满后2个交易日内报告并公告

### 6.4 司法强制执行披露
- 收到执行通知后2个交易日内披露
- 披露内容：拟处置股份数量、来源、方式、时间区间等

---

## 七、离婚导致股份变动（第10条）
- 股份过出方、过入方应当**持续共同遵守**本规则的有关规定

---

## 八、个人信息申报（第11条）

| 申报时点 | 申报内容 |
|---------|---------|
| 新上市公司申请股票初始登记时 | 姓名、职务、身份证号、证券账户、离任职时间等 |
| 新任董事/监事任职事项通过后2个交易日内 | 同上 |
| 新任高级管理人员任职事项通过后2个交易日内 | 同上 |
| 个人信息发生变化后2个交易日内 | 同上 |
| 离任后2个交易日内 | 同上 |
| 交易所要求的其他时间 | 同上 |

---

## 九、股份变动披露（第12条）

### 9.1 披露时间
- 自事实发生之日起2个交易日内报告并公告

### 9.2 披露内容
| 内容项目 | 说明 |
|---------|------|
| 本次变动前持股数量 | - |
| 本次股份变动的日期、数量、价格 | - |
| 本次变动后的持股数量 | - |
| 交易所要求的其他事项 | - |

---

## 十、窗口期限制（第13条）

董监高在下列期间**不得买卖本公司股票**：

| 窗口期类型 | 限制期间 |
|-----------|---------|
| 年度报告、半年度报告 | 公告前15日内 |
| 季度报告、业绩预告、业绩快报 | 公告前5日内 |
| 重大事件 | 自发生之日起至依法披露之日止 |
| 其他 | 交易所规定的其他期间 |

---

## 十一、公司监督责任（第14条）
- 上市公司应当制定专项制度，加强对董监高持有及买卖股份行为的监督
- 董事会秘书负责管理董监高的身份及所持股份数据
- 每季度检查董监高买卖股票的披露情况
- 发现违法违规的，应及时向证监会、交易所报告

---

## 十二、董监高责任（第15条）
- 应保证本人申报数据的及时、真实、准确、完整

---

## 十三、违规减持的法律责任

### 13.1 证监会监管措施（第16条）
- 责令购回违规减持股份并向上市公司上缴差价
- 监管谈话
- 出具警示函

### 13.2 行政处罚（第17条）

| 情形 | 处罚依据 |
|-----|---------|
| 在限制期限内转让股份（违反第4、13条） | 《证券法》第186条 |
| 超出规定比例转让股份（违反第5条） | 《证券法》第186条 |
| 未预先披露减持计划或披露不符合规定（违反第9条） | 《证券法》第186条 |
| 其他违法行为 | 《证券法》相关规定 |

### 13.3 市场禁入
- 情节严重的，可对有关责任人员采取证券市场禁入措施

---

## 十四、新旧规则衔接
- 本规则自公布之日起施行
- 《上市公司董事、监事和高级管理人员所持本公司股份及其变动管理规则》（证监会公告〔2022〕19号）同时废止

---

## 十五、董监高减持流程速查表

| 步骤 | 操作 | 时限 |
|-----|------|------|
| 第一步 | 确认是否处于窗口期或限制转让期 | 交易前 |
| 第二步 | 计算当年可转让股份数量（上年末持股×25%） | 交易前 |
| 第三步 | 披露减持计划 | 首次卖出前15个交易日 |
| 第四步 | 进行减持交易 | 计划期间内 |
| 第五步 | 披露减持结果 | 实施完毕或届满后2个交易日 |
| 第六步 | 变动后申报 | 事实发生日起2个交易日 |

---

*注：本规则为2024年发布版本，具体执行时请以最新官方文件为准。*
