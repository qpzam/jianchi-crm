# 上市公司股东减持股份规则摘要（批次7）

## 一、创业投资基金股东减持股份特别规定（2020年修订）

### 1.1 适用主体
- 在中国证券投资基金业协会备案的创业投资基金
- **私募股权投资基金参照适用**（2020年修订新增）

### 1.2 企业认定标准（三选一）
基金所投资的企业只需满足以下任意一项：

| 认定类型 | 标准要求 |
|---------|---------|
| 早期企业 | 首次接受投资时，企业成立不满60个月 |
| 中小企业 | 职工人数≤500人；年销售额≤2亿元；资产总额≤2亿元 |
| 高新技术企业 | 截至**IPO上市日**已取得高新技术企业证书 |

### 1.3 投资期限计算
- **起点**：投资金额累计达到300万元 或 投资金额累计达到投资该企业总投资额50%之日（以孰早者为准）
- **终点**：**发行人首次公开发行上市日**（2020年修订由"受理日"改为"上市日"）

### 1.4 集中竞价交易减持比例

| 投资期限 | 减持时间窗口 | 减持上限 |
|---------|------------|---------|
| 不满36个月 | 90日内 | ≤公司股份总数1% |
| 满36个月不满48个月 | 60日内 | ≤公司股份总数1% |
| 满48个月不满60个月 | 30日内 | ≤公司股份总数1% |
| 满60个月及以上 | **不受限制** | 不受比例限制 |

### 1.5 大宗交易减持比例

| 投资期限 | 减持时间窗口 | 减持上限 | 受让方锁定期 |
|---------|------------|---------|------------|
| 不满36个月 | 90日内 | ≤公司股份总数2% | 6个月 |
| 满36个月不满48个月 | 60日内 | ≤公司股份总数2% | 6个月 |
| 满48个月不满60个月 | 30日内 | ≤公司股份总数2% | 6个月 |
| 满60个月及以上 | **不受限制** | 不受比例限制 | **豁免锁定期** |

### 1.6 关键创新（2020年修订）
1. **简化适用标准**：从"双要求"改为"三选一"
2. **取消大宗交易受让方锁定期**：60个月以上投资期限的受让方豁免6个月锁定期
3. **私募股权基金参照适用**：扩大政策受惠面
4. **投资期限计算优化**：从"受理日"改为"上市日"，IPO排队期计入投资期限

## 二、大股东及特定股东减持规定（深交所第18号指引）

### 2.1 股东类型
- **大股东**：持股5%以上的股东、实际控制人
- **特定股东**：大股东以外持有IPO前股份的股东

### 2.2 不得减持的情形

#### 大股东不得减持情形：
1. 因涉嫌证券期货违法犯罪被立案调查/侦查/处罚/判刑未满6个月
2. 被交易所公开谴责未满3个月
3. 尚未足额缴纳罚没款（减持资金用于缴纳罚没款的除外）
4. 法律法规规定的其他情形

#### 控股股东/实控人不得减持情形：
1. 上市公司涉嫌违法犯罪被调查/处罚/判刑未满6个月
2. 上市公司可能触及重大违法强制退市情形期间
3. 上市公司被公开谴责未满3个月

#### 破发、破净、分红不达标限制（控股股东/实控人）：
1. **破发**：股价（向后复权）低于IPO发行价
2. **破净**：股价（向后复权）低于每股净资产
3. **分红不达标**：最近三年未分红或累计分红低于年均净利润30%

### 2.3 减持比例限制

| 减持方式 | 适用股东 | 时间窗口 | 减持上限 |
|---------|---------|---------|---------|
| 集中竞价 | 大股东、特定股东 | 90日内 | ≤公司股份总数1% |
| 大宗交易 | 大股东、特定股东 | 90日内 | ≤公司股份总数2% |

### 2.4 协议转让
- 受让方受让比例不得低于公司股份总数的5%
- 受让方在受让后6个月内不得减持
- 大股东减持后不再具有大股东身份的，出让方、受让方在6个月内仍需遵守减持比例限制

## 三、董监高减持规定（2022年修订）

### 3.1 不得转让情形
1. 公司股票上市交易之日起1年内
2. 离职后半年内
3. 承诺一定期限内不转让且在该期限内

### 3.2 年度减持比例限制
- 任职期间每年减持不得超过其所持本公司股份总数的25%
- 持股不超过1000股的，可一次全部转让

### 3.3 买卖股票禁止期间
1. 定期报告（年报、半年报）公告前30日内
2. 季报、业绩预告、业绩快报公告前10日内
3. 重大事项发生之日至披露之日
4. 交易所规定的其他期间

### 3.4 新增股份可转让数量
- 年内新增无限售条件股份当年可转让25%
- 新增有限售条件股份计入次年可转让股份计算基数

## 四、创业板询价转让和配售方式减持（2025年修订）

### 4.1 询价转让

#### 受让方条件：
- 具备定价能力和风险承受能力的机构投资者
- 符合创业板网下投资者条件的机构投资者
- 已在基金业协会完成登记的私募基金管理人

#### 转让数量要求：
- 单独或合计拟转让数量不得低于公司股份总数1%

#### 转让价格下限：
- 不低于发送认购邀请书之日前20个交易日股票交易均价的70%

#### 受让方减持限制：
- 受让股份后6个月内不得减持
- 该股份不属于特定股份，不适用特定股份减持规定

### 4.2 向股东配售

#### 配售比例要求：
- 单独或合计拟减持数量达到或超过公司股份总数5%的可采取配售

#### 配售价格下限：
- 不低于配售计划书公告日前20个交易日股票交易均价的70%

#### 配售对象：
- 股权登记日登记在册的上市公司其他股东（共同实施配售的股东除外）

### 4.3 重要特点
- 询价转让/配售减持数量**不计入**第18号指引规定的减持受控数量
- 受让方可为机构投资者（含其管理的产品）

## 五、股份锁定期安排

### 5.1 IPO股份锁定期

| 股份类型 | 持有人 | 锁定期 |
|---------|-------|--------|
| 控股股东、实控人 | - | 36个月 |
| 突击入股（主板） | 上市前一年从控股股东/实控人处受让或增资 | 36个月 |
| 突击入股（创业板） | 申请受理前6个月从控股股东/实控人处受让或增资 | 36个月 |
| 发起人及发行前股份 | - | 1年 |

### 5.2 非公开发行锁定期

| 持有人 | 锁定期 |
|-------|--------|
| 控股股东、实控人或其控制的关联方 | 36个月 |
| 通过认购取得实际控制权的投资者 | 36个月 |
| 战略投资者 | 36个月 |
| 其他发行对象 | 12个月 |

### 5.3 创业板非公开发行
- 发行价≥发行期首日前一个交易日均价：即可上市交易
- 均价90%≤发行价≤均值：锁定期12个月

## 六、监管红线与法律责任

### 6.1 违规行为
1. 弄虚作假骗取减持政策优待
2. 超出规定比例减持
3. 未按规定披露减持计划
4. 通过交易、转让或其他安排规避减持规定
5. 在禁止减持期限内减持

### 6.2 监管措施
1. 行政监管措施：警示函、责令改正、计入诚信档案
2. 交易所自律监管措施：书面警示、限制交易、通报批评、公开谴责
3. 证券市场禁入
4. 责令购回违规减持股份并上缴价差

### 6.3 禁止规避减持的方式
1. 融券卖出本公司股份
2. 转融通出借限制转让期股份
3. 开展以本公司股份为标的的衍生品交易
4. 通过ETF认购/申购变相减持
5. 通过离婚、解散、解除一致行动关系绕道减持
6. 通过司法强制执行、股票质押平仓、赠与方式减持

## 七、2024年减持新规与创投基金衔接

### 7.1 专门规定优先原则
《上市公司股东减持股份管理暂行办法》第26条：
> 中国证监会、证券交易所对创业投资基金、私募股权投资基金等减持股份另有规定的，从其规定。

### 7.2 创投基金豁免情形
创投基金的反向挂钩政策在2024年减持新规下持续有效，不受普适性禁令影响：
- 破发、破净、分红不达标限制（如非控股股东/实控人身份）
- 严格的减持比例限制

### 7.3 仍需遵守的规则
1. 禁止转融通出借、融券卖出
2. 禁止开展衍生品交易
3. 不得通过ETF变相减持
4. 必须遵守交易规则和披露要求

## 八、AMBERS系统申报指南

### 8.1 前置条件
1. 基金已在基金业协会备案
2. 产品季度信息更新已完成
3. 投资金额、投资时点已准确填报

### 8.2 申报流程（六步法）

| 步骤 | 内容 |
|-----|------|
| 第一步 | 登录AMBERS系统，进入"政策申请"模块 |
| 第二步 | 选择拟申请优惠政策的目标基金 |
| 第三步 | 填报拟减持标的企业名称、证券代码、企业性质（早期/中小/高企）、股票账户号、确权日期、上市日期 |
| 第四步 | 上传证明材料（承诺函、上市公告书、招股书、证券账户变动明细、企业性质证明） |
| 第五步 | 提交审核，等待机审+人工复核 |
| 第六步 | 审核通过后，基金对外公示信息页面添加"标识" |

### 8.3 审核时限
- 材料齐备且合规之日起**10个工作日**内完成审核

## 九、实战话术要点（来自QA汇总）

### 9.1 大宗交易优势话术
1. 避免提前发减持公告，3个交易日后即可启动
2. 可一次性大批量减持，节省时间和精力
3. 有利于盘面稳定，给投资者带来信心

### 9.2 应对常见异议

| 客户异议 | 应对策略 |
|---------|---------|
| 中介太多，你们优势？ | 从2015年专业做买断式大宗，合作上市公司1300+家 |
| 为什么不直接资金对接？ | 90%以上交易避免直接接触防范内幕交易 |
| 股价涨了3倍，不打算卖 | 强调这波行情不卖，锁定期满后只能减持1%，行情浪费 |
| 我们自己有渠道 | 国际大投行客户，北向通道投资，提升公司形象 |

### 9.3 折扣谈判要点
- 绿盘1个点内基本没问题，绿较多需增加折扣空间
- 量越大，折扣越有利
- 想要更多后端，折扣越低获利越多
