# 上市公司股东减持规则摘要（批次9）

## 一、创业板上市公司股东询价和配售方式转让股份业务指引（旧版）

### 1.1 适用范围
- 适用对象：深交所创业板上市公司股东
- 适用股份：首次公开发行前已发行股份（首发前股份）
- 适用方式：询价转让、配售方式转让

---

## 二、询价转让规则

### 2.1 询价转让股份锁定申报
| 项目 | 要求 |
|-----|------|
| 申报时间 | 询价转让计划书披露日15:00前 |
| 申报主体 | 出让股东通过托管券商向深交所申报 |
| 锁定数量 | 不得超过其所持无权利瑕疵的股份数量 |
| 锁定效力 | 锁定期间不得出质或转让，效力不及于孳息 |

### 2.2 询价转让流程

| 步骤 | 时间 | 操作 |
|-----|------|------|
| 第一步 | 计划书披露日15:00前 | 股东通过托管券商进行股份锁定申报 |
| 第二步 | 当日日终 | 中国结算实施股份锁定 |
| 第三步 | 询价业务T日15:00前 | 受托券商申报成交结果和自营交易单元号 |
| 第四步 | T日日终 | 解除锁定+办理过户登记+维护特定股份信息 |
| 第五步 | T+1日 | 中国结算从受托券商备付金账户扣收税费 |
| 第六步 | T日后 | 受托券商向出让股东划付所得款（扣除税费） |

### 2.3 询价转让失败处理
- 若过户失败，中国结算于T日日终通知受托券商
- 受托券商应重新确定和申报成交结果

---

## 三、向股东配售规则

### 3.1 配售登记信息申报
| 项目 | 要求 |
|-----|------|
| 申报时间 | 配售计划书披露日前一交易日收市后 |
| 申报主体 | 受托券商向深交所申报 |
| 申报内容 | 配售代码、配售股权登记日、过户日等 |

### 3.2 配售股份锁定申报
| 项目 | 要求 |
|-----|------|
| 申报时间 | 配售计划书披露日15:00前 |
| 申报主体 | 实施配售的股东通过托管券商向深交所申报 |
| 拟配售数量 | 不得超过其所持无权利瑕疵的股份数量 |

### 3.3 配售流程时间表

| 步骤 | 时间 | 操作 |
|-----|------|------|
| 第一步 | 计划书披露日前一交易日收市后 | 受托券商申报配售登记信息 |
| 第二步 | 计划书披露日15:00前 | 股东进行配售股份锁定申报 |
| 第三步 | 配售股权登记日 | 中国结算向其他股东分配配售权 |
| 第四步 | 配售登记日+5个交易日（配售业务T日） | 配售对象申购配售股份 |
| 第五步 | T日日终 | 解除锁定+办理过户登记+维护特定股份信息 |
| 第六步 | T+1日 | 结算税费（纳入多边净额结算） |

### 3.4 配售权分配原则
- 配售对象：登记在册的上市公司其他股东
- 配售比例：无限售条件流通股、有限售条件流通股按相同比例享有配售权
- 未足额行使配售权处理：实施配售的股东按认购总股份数占拟配售股份总数的比例进行配售

### 3.5 配售终止情形
- 在配售业务T日前出现需终止配售业务的情形
- 受托券商应及时向深交所提交配售终止申请
- 中国结算解除锁定股份，注销已生成的配售权

## 四、税收和费用

### 4.1 减持受控数量认定
- 通过询价或配售方式转让股份的数量，**不计入**《深圳证券交易所上市公司自律监管指引第18号》规定的减持受控数量

### 4.2 费用收取标准
| 费用类型 | 收取依据 |
|---------|---------|
| 经手费 | 按深交所费用标准收取 |
| 过户费 | 按中国结算费用标准收取 |
| 印花税 | 按国家有关规定执行 |
| 股息红利差别化个人所得税 | 股东依法履行纳税义务 |
| 个人转让限售股所得税 | 股东依法履行纳税义务 |

---

## 五、存托凭证参照适用
- 创业板上市存托凭证持有人以询价、配售方式转让其持有的存托凭证，参照适用本指引

---

## 六、责任划分

### 6.1 参与主体责任
- 投资者违反规定或托管券商、受托券商提交申报信息有误，由相应参与主体承担法律责任
- 深交所和中国结算可对违规主体采取自律监管措施或进行纪律处分

### 6.2 申报要求
- 托管券商、受托券商应按照投资者的真实意愿提交真实、准确、完整的业务申报

---
